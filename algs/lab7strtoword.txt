Условие: 
Цель функции: разбить текстовые строки на слова

		Алгоритм по шагам функции strtoword(char** p, int a, ..., int i)

	Начало функции
1. Объявление локальных переменных:
	beg - целочисленная переменная, позиция первой буквы слова в строке
	end - целочисленная  переменная, позиция последней буквы слова в строке
	j - целочисленный счётчик по i-ой строке массива p; 
	изначальное значение j = позиция последнего символа в i-ой строке
2. Если j = 1, то i--, переход к пункту 16.
3. Цикл счётчика j до 0.
4. Цикл перемещения j до позиции последней буквы в строке. 
5.  Присвоить end значение j.
6. Цикл перемещения j пока p[i][j] - буква.
7. Присвоить beg значение j.
8. Если beg = 0, переход к пункту 16.
9. Если beg > 0:
10. а++, перевыделить память массива указателей p по размеру а
11. Выделить под p[a-1] (end-beg+2)*sizeof(int) памяти 
12.  Цикл посимвольного копирования слова из p[i] в p[a-1]
13. Поставить терминаторы строк в p[i][beg], p[a-1][end-beg+1].
14.  Перевыделить память под строку p[i] по размеру (beg+1).
    Конец пункта 9.
15.  Присвоить значение a в ячейку памяти *A (возврат из функции)
16. Если i >= 0, strtoword (p, a, A, i).
17. Возврат p.
18. Конец
