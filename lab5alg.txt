Определить, является ли данная матрица латинским квадратом
(Латинский квадрат - квадратная матрица, каждая строка и каждый столбец которой длины n включают в себя все числа от 1 до n)
Цель функции - проверка двумерного динамического массива целых чисел на признак латинского квадрата

						Алгоритм по шагам функции latinf:
1. Начало
2. Объявление локальных переменных:
	целочисленные:
		счётчики i, j 
		check - проверка наличия всех чисел в строке\столбце 
		summ - счётчик от 1 до n для проверки наличия чисел
		num - параметр проверки чисел
3. Цикл по строкам при !check, обнуление summ
4. Цикл счётчика num от 1 до n при !check
5. Цикл проверки строки при !check
6. Конец цикла проверки строки
7. Проверка наличия всех чисел [1;n] в строке, иначе check++
8. Конец цикла num
9. Конец цикла по строкам
10. Цикл по столбцам при !check, обнуление summ
11. Цикл счётчика num от 1 до n при !check
12. Цикл проверки столбца при !check
13. Конец цикла проверки столбца 
14. Проверка наличия всех чисел [1;n] в столбце, иначе check++
15. Конец цикла num
16. Конец цикла по столбцам
17. Вывод результата в зависимости от значения check
18. Конец
