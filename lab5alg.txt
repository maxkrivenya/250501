Цель функции - проверка двумерного динамического массива целых чисел на признак латинского квадрата

	целочисленные:

		num - счётчик для поиска чисел от 1 до n в строке\столбце 

3. Цикл по строкам при !check
	4. Цикл счётчика num от 1 до n при !check
		5. Цикл проверки строки при !check

		7. Конец цикла проверки строки
		8. Проверка результата поиска текущего значения num в строке, иначе check++
	9. Конец цикла num
10. Конец цикла по строкам

11. Цикл по столбцам при !check
	12. Цикл счётчика num от 1 по n при !check
		13. Цикл проверки столбца при !check
		14. Сравнение текущего элемента столбца с num, выход из цикла при совпадении
		15. Конец цикла проверки столбца 
		16. Проверка результата поиска текущего значения num в столбце, иначе check++
	17. Конец цикла num
18. Конец цикла по столбцам

19. Вывод результата в зависимости от значения check
